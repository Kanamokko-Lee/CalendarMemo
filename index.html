設定手順
1. Google Cloud Console でプロジェクトを作成
2. 「Google Calendar API」を有効化
3. 認証情報 → API キーを作成（Calendar API に制限推奨）
4. 認証情報 → OAuth 2.0 クライアントID を作成（Webアプリ）
5. 承認済みオリジンに GitHub Pages の URL を追加
