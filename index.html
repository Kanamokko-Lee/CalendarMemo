⚙ SETTINGS

Google Calendar API を呼び出すための API キーです。

xxxxxxxx.apps.googleusercontent.com の形式で入力してください。

設定手順
1. Google Cloud Console でプロジェクトを作成
2. 「Google Calendar API」を有効化
3. 認証情報 → API キーを作成
4. 認証情報 → OAuth 2.0 クライアントID を作成（ウェブアプリ）
5. 承認済みオリジンに GitHub Pages の URL を追加
⚠ 入力値はブラウザの localStorage にのみ保存されます。
